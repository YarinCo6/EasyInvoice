## 📖 שימוש

### הגדרת פרטי עסק
```
1. לחץ על "פרטי עסק"
2. העלה לוגו (אופציונלי)
3. מלא את כל השדות
4. לחץ "שמור פרטים"
```

### יצירת קבלה
```
1. לחץ על "יצירת קבלה"
2. מלא פרטי לקוח ושירות
3. הזן סכום ואמצעי תשלום
4. לחץ "צור קבלה"
```

### הורדת PDF
```
1. לאחר יצירת הקבלה
2. לחץ על כפתור ההורדה (⬇)
3. הקובץ יישמר אוטומטית
```
